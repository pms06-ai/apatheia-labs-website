## Contributing Tutorials

Community contributions welcome. If you'd like to create a tutorial:

**Content guidelines**:
- Focus on practical application
- Use real examples (properly redacted) or realistic simulations
- Include common pitfalls and solutions
- Reference official documentation
- Maintain professional production quality

**Technical requirements**:
- Minimum 1080p resolution
- Clear audio (use good microphone)
- Edited for pacing (no long pauses, errors, or tangents)
- Captions/subtitles (accessibility)
- Consistent branding

**Submission process**:
1. Propose tutorial topic (GitHub Discussions)
2. Get feedback and approval
3. Create tutorial following guidelines
4. Submit for review
5. Incorporate feedback
6. Publish with attribution
